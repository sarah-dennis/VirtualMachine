YAAL Examples

Below are some example programs that can help you learn the YAAL language. Feel free to copy and paste these text files into the text editor.

  • Graphic and Input Handling Examples

    • ClickGame

    A game demonstrating how to handle mouse clicks.

    Goal

    A graphics game demonstrating how to handle keyboard input.

    Etch-A-Sketch

    A graphics program that handles mouse and key input.

  • Timer Examples

    • Countdown

    A simple program that counts down from a specified number, showing how to schedule events.

    Get-Faster-Countdown

    A variation of Countdown that shows a more advanced use of timers.

  • Math and Calculation Examples

    • GCD

    A program that calculates the GCD of two given numbers. Demonstrates simple math operations.

    Recursive Sum

    A program that adds all the numbers less than or equal to a given number, demonstrating recursion.

    ModExp

    A program that calculates the remainder of one number raised to the power of another number and divided by another number. Demonstrates the calculation power of the language.

    Quicksort

    A program that sorts a randomly populated array using the quicksort algorithm.

    Prime Sieve

    A program that calculates the number of primes less than a given number, based on the Sieve of Eransthoses.

    Visual Prime Sieve

    A variant of the Prime Sieve utilizing graphics.
